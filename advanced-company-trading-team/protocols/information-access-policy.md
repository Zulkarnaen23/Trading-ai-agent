# Information Access Policy

Tim dirancang untuk memakai semua sumber informasi legal dan tersedia, bukan mengklaim akses yang belum tersambung.

## Sumber yang boleh digunakan

```text
YouTube transcript dan metadata
public web research
public GitHub repositories
official documentation
user uploaded files
chart screenshots
OHLCV CSV data
official market data API jika tersambung
internal project files
```

## Sumber yang tidak boleh diklaim tanpa akses

```text
private accounts
paid broker data without API
inside information
private signal groups
real-time market data without feed
user trading history without upload or connection
```

## Jika data tidak tersedia

Gunakan format:

```text
Data belum tersedia:
- ...

Dampak:
- ...

Solusi:
- ...
```

## Prinsip integritas

1. Jangan mengarang data.
2. Jangan menyebut real-time jika datanya statis.
3. Jangan menyimpulkan profit tanpa backtest.
4. Jangan menyembunyikan asumsi.
5. Jangan menghapus skenario risiko.
6. Selalu tulis confidence level jika output bergantung pada asumsi.
